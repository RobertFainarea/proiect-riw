Iasi

Iasi (livresc Iasii, respectiv Targu' Iesilor) este municipiul de resedinta al judetului cu acelasi nume, Moldova, Romania. Iasiul a fost capitala Moldovei in perioada 1564-1859, una dintre cele doua capitale ale Principatelor Unite intre 1859 si 1862 si capitala Regatului Romaniei intre 1916 si 1918 (in perioada Primului Razboi Mondial, atunci cand Bucurestiul s-a aflat sub ocupatie germana).
