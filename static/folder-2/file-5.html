Botoșani (germană Bottuschan) este municipiul de reședință al județului cu același nume, Moldova, România. Este așezat în partea de sud-vest a județului Botoșani, pe interfluviul dintre râurile Sitna și Dresleuca, spre vest între Dresleuca și Siret, apoi coboară între dealurile Crivăț, Agafton, Baisa, în adâncuri sprijinindu-se pe platforma Moldovei.
