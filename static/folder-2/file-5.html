Botosani

Botosani (germana Bottuschan) este municipiul de resedinta al judetului cu acelasi nume, Moldova, Romania. Este asezat in partea de sud-vest a judetului Botosani, pe interfluviul dintre raurile Sitna si Dresleuca, spre vest intre Dresleuca si Siret, apoi coboara intre dealurile Crivat, Agafton, Baisa, in adancuri sprijinindu-se pe platforma Moldovei.
