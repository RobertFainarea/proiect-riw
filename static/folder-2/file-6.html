Vaslui

Vaslui este municipiul de resedinta al judetului cu acelasi nume, Moldova, Romania, format din localitatile componente Bahnari, Brodoc, Moara Grecilor, Rediu, Vaslui (resedinta) si Viisoara. Se afla in estul Romaniei, aproape de frontiera cu Republica Moldova. Este atestat documentar in anul 1375. Municipiul Vaslui este membru al Conventiei primarilor pentru energie locala durabila din anul 2010.
