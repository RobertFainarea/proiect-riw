Vaslui este municipiul de reședință al județului cu același nume, Moldova, România, format din localitățile componente Bahnari, Brodoc, Moara Grecilor, Rediu, Vaslui (reședința) și Viișoara. Se află în estul României, aproape de frontiera cu Republica Moldova. Este atestat documentar în anul 1375. Municipiul Vaslui este membru al Convenției primarilor pentru energie locală durabilă din anul 2010.
